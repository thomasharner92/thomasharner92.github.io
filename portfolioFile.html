Work Samples


Bike Tour Map

Details of a fictional bike race through the Southwestern United States and up the California coast. Terrain created using Photoshop, map styling done via Adobe Illustrator.

Climate Change Infographic

Study of the increase in US temperature by county in the past 30 years, relative to each county's opinion on the impacts of climate change. Created using Adobe Illustrator.

Crowdsourced Data Comp.

Slider web map detailing the differences in crowd-sourced OpenStreetMap data for three Virginia counites relative to local government. Created with MapboxGL.

MLB Team Performance

Temporal web map detailing wins relative to team attendance for each Major League Baseball team from 2011-2016. Created with Leaflet.

Sensationalist Map

A map the uses visually striking, emotive imagery in order to appeal to american citizens to support drone strikes across the Middle East. Created using Adobe Illustrator.

D3 Project

A project testing the D3 library for data visualization of New Zealand sample data using vector data with dynamic labels and charts.
