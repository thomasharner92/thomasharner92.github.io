Fairfax County Data
Editing Application

Web Application that provides Fairfax County officials the ability to use various widgets within the ArcGIS JavaScript API to compare and edit their data to OpenStreetMap data via viewing Feature Services.

Letchworth State Park
Visitors Guide

Web Application that provides a guided tour of Letchworth State Park. Uses Postgres, GeoServer, and OpenLayers for trail querying and WFS feature editing to update status reports on features throughout the park.

Nitrate Interpolation Application

Web Application that uses Java, Postgres, GeoServer, and OpenLayers to dynamically build hexbins to interpolate cancer rates for nitrate concentrations throughout the state of Wisconsin.
